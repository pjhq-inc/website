# PJHQ Website

Welcome to the official repository for the **PJHQ website**! This project is built with **React** to deliver a dynamic, modern, and user-friendly experience.

---

## 🌐 Live Website
Check out the live website here: [https://pjhq.dev](https://pjhq.dev)

---

## 🚀 Features
- **Built with React:** A fast, component-based front-end library.
- **Responsive Design:** Looks great on all devices (desktop, tablet, mobile).
- **About Us Page:** Learn more about our mission, vision, and team.
- **Cookie Compliance:** Fully compliant with GDPR and other privacy regulations.
- **Easy to Maintain:** Modular components make updates a breeze.

---

## 🛠️ Technologies Used
- **Frontend:**
  - [React](https://reactjs.org) (JavaScript library for building UIs)
  - [React Router](https://reactrouter.com) (for navigation)
  - [Bootstrap](https://getbootstrap.com) (for styling)
  - [CSS Modules](https://github.com/css-modules/css-modules) (for scoped styling)
- **Build Tool:**
  - [Vite](https://vitejs.dev) (fast build tool for modern web apps)
- **Hosting:**
  - Hosted on [cPanel](https://cpanel.net)
  - Domain: `pjhq.dev`
- **Cookie Management:**
  - Custom cookie banner for GDPR compliance.
